Java Study Material

Important functions and concepts for quick reference

1. String Functions (20 Examples)

1. split(delimiter)

➤ Splits a string into parts wherever the given delimiter occurs. Useful for breaking CSV strings or user input into parts.


String[] parts = "a,b,c".split(",");
System.out.println(parts[1]);

Output: b

2. charAt(index)

➤ Returns the character present at the specified index. Index starts from 0.


String s = "Java";
System.out.println(s.charAt(2));

Output: v

3. substring(start, end)

➤ Extracts the substring from start index to (end-1). Does not include the character at end index.


String s = "Programming";
System.out.println(s.substring(0, 6));

Output: Progra

4. equals(str)

➤ Compares two strings for exact match (case-sensitive).


System.out.println("java".equals("java"));

Output: true

5. equalsIgnoreCase(str)

➤ Compares two strings ignoring case. Useful for user input comparison.


System.out.println("Java".equalsIgnoreCase("java"));

Output: true

6. length()

➤ Returns the number of characters present in the string.


String s = "Hello";
System.out.println(s.length());

Output: 5

7. toLowerCase()

➤ Converts all characters in the string to lowercase.


System.out.println("JAVa".toLowerCase());

Output: java

8. toUpperCase()

➤ Converts all characters in the string to uppercase.


System.out.println("hello".toUpperCase());

Output: HELLO

9. trim()

➤ Removes any leading and trailing spaces from the string.


System.out.println(" Java ".trim());

Output: Java

10. contains(str)

➤ Checks if the specified string exists inside another string.


System.out.println("welcome".contains("come"));

Output: true

11. indexOf(char)

➤ Returns the first index of the specified character. Returns -1 if not found.


System.out.println("hello".indexOf('l'));

Output: 2

12. lastIndexOf(char)

➤ Returns the last index where the character appears.


System.out.println("hello".lastIndexOf('l'));

Output: 3

13. replace(old, new)

➤ Replaces every occurrence of a character or string with a new one.


System.out.println("apple".replace('p', 'b'));

Output: abble

14. startsWith(prefix)

➤ Checks if the string starts with the given prefix.


System.out.println("program".startsWith("pro"));

Output: true

15. endsWith(suffix)

➤ Checks if the string ends with the given suffix.


System.out.println("notebook".endsWith("book"));

Output: true

16. isEmpty()

➤ Returns true if the string is empty (has 0 characters).


System.out.println("".isEmpty());

Output: true

17. concat(str)

➤ Joins the specified string to the end of another string.


System.out.println("Hello ".concat("World"));

Output: Hello World

18. compareTo(str)

➤ Compares two strings lexicographically (alphabetical order).


System.out.println("abc".compareTo("abd"));

Output: -1

19. toCharArray()

➤ Converts a string into a character array.


char[] chars = "Hi".toCharArray();
System.out.println(chars[1]);

Output: i

20. format()

➤ Returns a formatted string using placeholders (like printf).


String name = "John";
System.out.println(String.format("Hello %s", name));

Output: Hello John

2. Array Functions (20 Examples)

1. length

➤ Returns the total number of elements in the array. It's a property, not a method.


int[] arr = {1, 2, 3};
System.out.println(arr.length);

Output: 3

2. Arrays.toString()

➤ Converts an array to a readable string format using Arrays utility class.


int[] arr = {5, 10, 15};
System.out.println(Arrays.toString(arr));

Output: [5, 10, 15]

3. Arrays.sort()

➤ Sorts the elements of the array in ascending order. It modifies the original array.


int[] arr = {30, 10, 20};
Arrays.sort(arr);
System.out.println(Arrays.toString(arr));

Output: [10, 20, 30]

4. Arrays.equals()

➤ Compares two arrays and returns true if both contain the same elements in order.


int[] a = {1, 2};
int[] b = {1, 2};
System.out.println(Arrays.equals(a, b));

Output: true

5. Arrays.fill()

➤ Fills the array with a specific value. Useful for initializing arrays quickly.


int[] arr = new int[3];
Arrays.fill(arr, 7);
System.out.println(Arrays.toString(arr));

Output: [7, 7, 7]

6. Arrays.copyOf()

➤ Copies the first n elements into a new array. Extra elements will be 0 or null.


int[] arr = {1, 2, 3, 4};
int[] copy = Arrays.copyOf(arr, 2);
System.out.println(Arrays.toString(copy));

Output: [1, 2]

7. Arrays.copyOfRange()

➤ Copies elements in the range from start index to end index - 1 into a new array.


int[] arr = {10, 20, 30, 40};
int[] range = Arrays.copyOfRange(arr, 1, 3);
System.out.println(Arrays.toString(range));

Output: [20, 30]

8. Arrays.binarySearch()

➤ Searches for a value in a sorted array. Returns the index or a negative value if not found.


int[] arr = {2, 4, 6, 8};
int index = Arrays.binarySearch(arr, 6);
System.out.println(index);

Output: 2

9. Arrays.setAll()

➤ Sets array values using a lambda expression. Works well for logic-based initialization.


int[] arr = new int[5];
Arrays.setAll(arr, i -> i * i);
System.out.println(Arrays.toString(arr));

Output: [0, 1, 4, 9, 16]

10. Arrays.stream()

➤ Converts an array into a stream for operations like sum, average, etc.


int[] arr = {1, 2, 3};
int sum = Arrays.stream(arr).sum();
System.out.println(sum);

Output: 6

11. Arrays.asList()

➤ Converts an array into a List (fixed-size backed list).


String[] arr = {"A", "B"};
List list = Arrays.asList(arr);
System.out.println(list);

Output: [A, B]

12. for-each loop

➤ A simple way to iterate through all elements in an array.


int[] arr = {10, 20, 30};
for (int val : arr) {
  System.out.print(val + " ");
}

Output: 10 20 30

13. Multidimensional Array

➤ An array of arrays. Useful for representing tables or grids.


int[][] arr = {{1, 2}, {3, 4}};
System.out.println(arr[1][1]);

Output: 4

14. Clone Array

➤ Creates a shallow copy of the array.


int[] arr = {5, 6};
int[] clone = arr.clone();
System.out.println(Arrays.toString(clone));

Output: [5, 6]

15. Index Access

➤ Accessing values by index is constant time. Indexing starts at 0.


int[] arr = {9, 8, 7};
System.out.println(arr[1]);

Output: 8

16. Update Value

➤ You can directly update values at any index using assignment.


int[] arr = {1, 2, 3};
arr[0] = 100;
System.out.println(Arrays.toString(arr));

Output: [100, 2, 3]

17. Reverse Array (Manual)

➤ Reverse using loop to swap elements from both ends.


int[] arr = {1, 2, 3};
for(int i = 0; i < arr.length / 2; i++) {
  int temp = arr[i];
  arr[i] = arr[arr.length - 1 - i];
  arr[arr.length - 1 - i] = temp;
}
System.out.println(Arrays.toString(arr));

Output: [3, 2, 1]

18. Array Initialization

➤ Arrays can be initialized directly with values.


int[] arr = {11, 22, 33};
System.out.println(arr[0]);

Output: 11

19. Passing Array to Method

➤ Arrays are passed by reference. Changes in method affect original array.


void modify(int[] a) {
  a[0] = 99;
}
int[] arr = {1, 2};
modify(arr);
System.out.println(arr[0]);

Output: 99

20. Array of Objects

➤ You can create arrays of any type, including objects like String, Integer, etc.


String[] names = new String[2];
names[0] = "Ram";
names[1] = "Sam";
System.out.println(names[1]);

Output: Sam

3. Math Functions (20 Examples)

1. Math.max(a, b)

➤ Returns the larger of two values. Useful for comparisons like max score, max value etc.

System.out.println(Math.max(10, 20));

Output: 20

2. Math.min(a, b)

➤ Returns the smaller of two numbers. Helps find the minimum in comparisons.

System.out.println(Math.min(5, 9));

Output: 5

3. Math.abs(x)

➤ Returns the absolute (positive) value of a number. Useful for distance calculations.

System.out.println(Math.abs(-8));

Output: 8

4. Math.sqrt(x)

➤ Returns square root of the number. Result is of type double.

System.out.println(Math.sqrt(25));

Output: 5.0

5. Math.pow(a, b)

➤ Returns the value of a raised to the power of b (a^b).

System.out.println(Math.pow(2, 3));

Output: 8.0

6. Math.floor(x)

➤ Rounds down the decimal number to the nearest whole number.

System.out.println(Math.floor(3.8));

Output: 3.0

7. Math.ceil(x)

➤ Rounds up the decimal to the next highest whole number.

System.out.println(Math.ceil(3.2));

Output: 4.0

8. Math.round(x)

➤ Rounds the decimal to the nearest whole number. Uses .5 as cutoff.

System.out.println(Math.round(2.6));

Output: 3

9. Math.random()

➤ Returns a random double value between 0.0 (inclusive) and 1.0 (exclusive).

System.out.println(Math.random());

Output: Varies (e.g., 0.5278)

10. Math.cbrt(x)

➤ Returns the cube root of a number.

System.out.println(Math.cbrt(27));

Output: 3.0

11. Math.toRadians(degree)

➤ Converts degree angle to radians. Used in trigonometry.

System.out.println(Math.toRadians(180));

Output: 3.141592653589793

12. Math.toDegrees(radian)

➤ Converts radians into degrees.

System.out.println(Math.toDegrees(Math.PI));

Output: 180.0

13. Math.sin(x)

➤ Returns the sine of the angle (in radians).

System.out.println(Math.sin(Math.toRadians(90)));

Output: 1.0

14. Math.cos(x)

➤ Returns the cosine of an angle in radians.

System.out.println(Math.cos(Math.toRadians(0)));

Output: 1.0

15. Math.tan(x)

➤ Returns the tangent of an angle in radians.

System.out.println(Math.tan(Math.toRadians(45)));

Output: 1.0

16. Math.log(x)

➤ Returns natural logarithm (base e) of a number.

System.out.println(Math.log(1));

Output: 0.0

17. Math.log10(x)

➤ Returns logarithm to base 10 of a number.

System.out.println(Math.log10(100));

Output: 2.0

18. Math.exp(x)

➤ Returns Euler’s number e raised to the power of x.

System.out.println(Math.exp(2));

Output: 7.38905609893065

19. Math.signum(x)

➤ Returns -1.0 if x is negative, 0.0 if zero, 1.0 if positive.

System.out.println(Math.signum(-99));

Output: -1.0

20. Math.copySign(mag, sign)

➤ Copies sign of second number to first number (magnitude).

System.out.println(Math.copySign(6.5, -1));

Output: -6.5
